Motion Planning Library for Autonomous Navigation

CommonRoad is a collection of composable benchmarksfor motion planning on roads, which provides researchers witha means of evaluating and comparing their motion planners.There are different planners that can be used to solve theplanning problems in ConmmonRoad scenarios: high-levelroute planner that finds sequences of lanelets, sampling-basedreactive planner, search-based planner. Other helpful packageshave also been developed such as commonroad-io, set-basedprediction, drivability checker, curvilinear coordinate systemsand so on.

  • About CommonRoad

The CommonRoad Motion Planning Library (MPL) integrates the existing CommonRoad packages to build a library that can access all its components and configure them. A task-specific motion planner can be simply created by setting up the configuration files. MPL can be easily extended based on the well-organized architecture. Also, it can be adapted to plan a trajectory for Carla or SUMO simulation. With easy installation and user manuals, it is very convenient and efficient to create a desired motion planner using MPL.
